## 📖 Project Overview
This project is a fully functional **Contact Form** built with **Next.js 15+, Tailwind CSS 4, and NodeMailer**.  
It supports **file uploads, spam protection, modals, and responsive UI**.

---

## 🛠 Technologies Used
- **Next.js 15+** – Full-stack React framework with API Routes  
- **Tailwind CSS 4** – Utility-first CSS framework  
- **NodeMailer** – Email sending via SMTP  
- **React Hooks** – State management and form handling  

---

## 🚀 Features
- 📧 Email integration with SMTP  
- 📂 File upload support  
- 🛡 Honeypot spam protection  
- ✅ Frontend + Backend validation  
- 🎉 Success & Error modals with timestamp  
- 📱 Fully responsive UI
